# HTML Brand — Examples

## Contents

| File | Format | Doc Type | Description |
|------|--------|----------|-------------|
| `sample-output.md` | Markdown | Executive | Component reference guide showing all Design System v5 elements |
| `sample-output.html` | HTML | Executive | Complete branded HTML page — hero, TOC, headers, tables, callouts, cards, Mermaid, score bars |
| `sample-carousel.html` | HTML | Carousel | 9-slide horizontal carousel with dot nav, swipe, keyboard ← →, progress bar, CTA slide |

## Usage

These examples demonstrate the full output of the `sofka-html-brand` skill for an executive-type document.

**Scenario:** Acme Financial — Core Banking Modernization executive summary. Demonstrates all Sofka Design System v4 components in a realistic deliverable context.

## Key Components Demonstrated

### Executive (sample-output.html)
1. **Hero Section** — Black background, orange border, 4 KPIs, `sofka_` logotype
2. **Sticky TOC** — Horizontal scroll navigation with active section tracking
3. **Numbered Sections** — 01-08 pattern with section headers
4. **Data Tables** — Styled with brand tokens, sticky headers
5. **Callout Cards** — Key insight (blue), warning (amber), success (yellow), critical (red)
6. **Score Bars** — Progress indicators with percentage labels
7. **Mermaid Diagrams** — Architecture diagrams with Sofka high-contrast theme
8. **Footer** — Status badges with document metadata

### Carousel (sample-carousel.html)
1. **Horizontal slides** — 100vw per slide, CSS transform navigation
2. **Dot navigation** — Active dot extends to pill shape
3. **Swipe support** — Touch events with 50px threshold
4. **Keyboard navigation** — ← → Home End
5. **Progress bar** — Top gradient orange→gold
6. **`sofka_` brand** — Logotype top-left, meta badges on hero
7. **CTA slide** — Final slide with action button + mailto link
8. **Print CSS** — Each slide becomes a page

### Presentation Slides (sample-presentation-slides.html)
1. **Full-screen slides** — Fade-in/slide transitions
2. **Arrow navigation** — ← → Space PageUp PageDown
3. **Speaker notes** — Toggle with N key, panel slides up from bottom
4. **Fullscreen mode** — F key toggles browser fullscreen
5. **Section dividers** — Large number + title pattern
6. **Progress bar** — Top gradient with percentage
7. **`sofka_` brand** — Logotype top-left, meta badges
8. **25 slides** — Full discovery walkthrough structure
